# upon

A tiny template engine.

## Features

- Configurable template delimiters: `<? value ?>`
- Rendering values: `{{ user.name }}`
- Conditionals: `{% if user.enabled %} User is enabled {% endif %}`
- Customizable filter functions: `{{ value | my_filter }}`
